# My GitHub Portfolio

Next.js (App Router) 기반의 현대적이고 깔끔한 개발자 포트폴리오 사이트입니다.

## 🚀 시작하기

먼저 의존성 패키지를 설치합니다:

```bash
npm install
# 또는
yarn install
```

개발 서버를 실행합니다:

```bash
npm run dev
# 또는
yarn dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

## 🛠 기술 스택

- **Frontend**: Next.js 14 (App Router), React 18
- **Styling**: Vanilla CSS (CSS Variables)
- **Deployment**: Vercel (권장)

## 📁 주요 폴더 구조

- `src/app`: 페이지 라우팅 및 전역 스타일
- `src/components`: 재사용 가능한 UI 컴포넌트
- `.env.local`: API 키 등 환경 변수 설정 (직접 생성 필요)
